# NovariaMC

NovariaMC is an Android app for creating, managing, and hosting Minecraft servers directly from your phone.

It includes a local control panel, server files, backups, world tools, performance monitoring, and support for Java and Bedrock-compatible workflows.

## Highlights

- Create and manage Minecraft servers from Android.
- Open a local web panel from a PC on the same Wi-Fi.
- Manage files, worlds, backups, mods, plugins, and server settings.
- Monitor RAM, CPU, temperature, battery, uptime, and connected players.
- Import and convert worlds with integrated tools.
- Use Playit for public access when configured.
