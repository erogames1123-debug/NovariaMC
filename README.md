# NovariaMC

NovariaMC is an Android app for creating, managing, and hosting Minecraft servers directly from a phone.

It is built for people who want a practical server manager on Android: local server control, a PC web panel, file management, backups, world tools, runtime handling, performance monitoring, and public access helpers in one place.

> NovariaMC is currently in active development. Features, compatibility, and interface details may change as the project evolves.

## Features

- Create and manage Minecraft server environments from Android.
- Start, stop, restart, and monitor servers from the app.
- Open a local web control panel from a PC on the same Wi-Fi network.
- Manage server files, worlds, backups, mods, plugins, and configuration.
- Monitor RAM, CPU, temperature, battery, uptime, and connected players.
- Import Bedrock `.mcworld` worlds and use integrated conversion tools where supported.
- Configure Playit for public access.
- Check for app updates through GitHub Releases.

## Development Status

NovariaMC is usable, but it is not final.

The current focus is improving stability, server compatibility, Android runtime support, UI polish, automation tools, and the local web console experience.

Some advanced features are experimental and may require additional testing on different devices.
