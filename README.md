# NovariaMC

NovariaMC es una app de Android para crear, gestionar y alojar servidores de Minecraft directamente desde un telefono.

Esta pensada para personas que quieren un gestor practico de servidores en Android: control local del servidor, panel web para PC, gestion de archivos, copias de seguridad, herramientas de mundos, manejo de runtimes, monitoreo de rendimiento y ayudas para acceso publico en un solo lugar.

> NovariaMC esta actualmente en desarrollo activo. Las funciones, la compatibilidad y los detalles de la interfaz pueden cambiar conforme el proyecto evoluciona.

## Funciones

- Crear y gestionar entornos de servidores Minecraft desde Android.
- Iniciar, detener, reiniciar y monitorear servidores desde la app.
- Abrir un panel de control web local desde una PC en la misma red Wi-Fi.
- Gestionar archivos del servidor, mundos, copias de seguridad, mods, plugins y configuracion.
- Monitorear RAM, CPU, temperatura, bateria, tiempo encendido y jugadores conectados.
- Importar mundos Bedrock `.mcworld` y usar herramientas de conversion integradas cuando sea compatible.
- Configurar Playit para acceso publico.
- Buscar actualizaciones de la app mediante GitHub Releases.

## Estado de desarrollo

NovariaMC es usable, pero no es final.

El enfoque actual es mejorar la estabilidad, compatibilidad de servidores, soporte de runtimes en Android, pulido de la interfaz, herramientas de automatizacion y la experiencia del panel web local.

Algunas funciones avanzadas son experimentales y pueden requerir pruebas adicionales en distintos dispositivos.
